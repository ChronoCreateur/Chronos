# ChronoCréateur

ChronoCréateur est une application web statique en français pour créer, modifier, sauvegarder et exporter des frises chronologiques . Inspirée de `friseschronos.fr` Le code a été vérifié par une IA locale (**Qwen2.5-Coder 32B**) afin de détecter d'éventuelles erreurs.
